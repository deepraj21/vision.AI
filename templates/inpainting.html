Inpainting

Requirement:

Image_file is the original image to process.
The original image should be a JPG or a PNG, with a maximum resolution of 16 megapixels and a max file size of 30 Mb. mask_file is the mask image, defining the areas that need to be removed. The mask image should be a PNG, and should have the same resolution as the original image and a max file size of 30 Mb.
The mask should be black and white with no grey pixels (e.g. values of only 0 or 255), the value of 0 indicating a pixel to keep as is and 255 a pixel to 'clean up'.
